DESCRIPCIÓN ACTIVIDADES SEMANA 11:

1) 	Diseñar a mano una interfaz gráfica de usuario (GUI) en una hoja u hojas de papel. Tomar una fotografía a cada hoja y guardarla
	como una imagen .png, con la siguiente plantilla de archivo:
		
		foto_diseno_1.png
		foto_diseno_2.png
		...
		foto_diseno_N.png
		
	Importante: Si sólo emplea una hoja, sólo tendrá que guardar una imagen con el nombre foto_diseno.png.
	
	El diseño debe contener un dibujo de cómo quedará la interfaz.
	
	El diseño corresponde al prototipo o idea inicial de la interfaz gráfica que tendrá el proyecto final de la asignatura.
	
	Importante: Como se trabaja en grupos, un proyecto será para dos personas, sin embargo, cada persona debe tener inicialmente
	un entendimiento personal, propio del proyecto. No se aceptan interfaces iguales para dos miembros de un mismo grupo, cada
	uno debe realizar su propio entendimiento y diseño del problema y por ende la interfaz. Es una actividad individual.
	
2) Implementar el diseño realizado en el Designer de PyQt, como se mostró en clase. Guardar el archivo de diseño de interfaz como:
	
		nombres_apellidos_diseno.ui
	
	Importante: Todos los componentes de la interfaz deben tener un nombre asociado. Esto se define con click derecho "Change Object Name...".
	El nombre debe ser lo suficientemente descriptivo como para saber el objetivo del widget. Sólo caracteres ASCII y separación de palabras
	con underscore.
	
3) Guardar las imágenes .png y el archivo .ui en un directorio con el nombre:

		nombres_apellidos
		
	Sólo se aceptan caracteres ASCII y minúsculas.

4)	Realizar un PR (Pull-Request) con el directorio creado anteriormente. El título del PR debe tener la siguiente plantilla:

		Semana 11 – Nombre Completo
		
	En la descripción del PR escribir:
	
		Actividades desarrolladas para la semana 11 por Nombre Completo. Se incluye diseño de prototipo de GUI de proyecto final 
		de asignatura.
		
	Los nombres y apellidos deben ser CamelCase es decir, mayúsculas iniciales. Para el título sólo se aceptan caracteres ASCII. 
	La eñe (ñ) reemplazarla con n, sin tildes. Las palabras separadas por espacios. La descripción puede tener eñes (ñ) y tildes.
